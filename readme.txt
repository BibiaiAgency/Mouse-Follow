== Description ==

Mouse Follow is a WordPress plugin developed by Bibiai that adds an elegant pointer to your website. With this tool you can give a unique touch to your site and draw the attention of your visitors.

== Installation ==

1. Visit `Plugins > Add New`
2. Search for `Mouse Follow`
3. Install Mouse Follow once it appears
4. Activate Mouse Follow from your Plugins page.
5. Tha's everything, just enjoy!

= After activation =

Once activated, Mouse Follow will automatically add the pointer to your site.
